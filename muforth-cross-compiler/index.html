MuFORTH cross compiler

Cross-compiler? Meta-compiler? Target-compiler? What’s the best term to use here?

It’s confusing because they make a nested set of containers (parallelogram, rectangle, square). I would say this: meta- and target-compilers are both cross-compilers; but a meta-compiler targets the same machine it runs on, while a target-compiler targets a different machine.

But you could also say this: meta- and cross- are target-compilers; but the meta-compiler targets the same (host) machine, while a cross compiler targets a different (target) machine.

Hence my confusion over terms.
